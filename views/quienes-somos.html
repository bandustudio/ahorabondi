Quienes somos

AhoraBondi es una red voluntaria de visualización de transporte público en tiempo real.

Objetivo

Somos un grupo de vecinos que nos establecimos el objetivo de lograr monitorear el tráfico de transporte público de colectivo en tiempo real. Para esto sabemos que debemos poner a disposición las herramientas y la tecnología necesaria a los empresarios del transporte público para que éstos contribuyan al mejoramiento de servicio poniendo a disposición la ubicación de sus unidades en tiempo real.

¿Cuál sería el beneficio?

El equipo de AhoraBondi usa servicios gratuitos para poner en funcionamiento el servidor. No cobramos nada a nadie, ni tenemos pensado cobrar nada a nadie en el futuro. El beneficio que aporta el proyecto es para los pasajeros, mostrando la ubicación real de cada unidad en un tiempo determinado, y también para los empresarios que podrán monitorear sus unidades para tomar mejores desiciones.

¿En qué estado se encuentra el proyecto actualmente?

Actualmente AhoraBondi está en etapa de pruebas. Las primeras pruebas fueron exitosas y esperamos en breve lograr la primera versión del proyecto.

¿Qué se necesita para lanzar el proyecto?

El aporte necesario para iniciar el proyecto es contar con un smartphone con GPS por cada unidad y cubrir el costo del paquete de datos necesario para transmitir en directo la ubicación mientras dure el tiempo de servicio de la unidad.

Necesitamos de la buena voluntad de al menos una empresa de transporte público para comenzar.

¿Qué tencología usa el proyecto?

AhoraBondi está construido sobre tecnología NODE JS, trabaja con una base de datos MONGO DB y usa Javascript para detectar y enviar las coordenadas GPS.

AhoraBondi no guarda datos de la actividad de ninguna empresa, solo se limita a identificar y nombrar a las unidades y transmitir su posición en tiempo real via WebSocket directamente al usuario.

¿Por qué "voluntario"?

Los que formamos actualmente el equipo de AhoraBondi consideramos indispensable la buena voluntad de los empresarios de transporte público, choferes y pasajeros para que este proyecto se realice de forma exitosa. Creemos que cualquier iniciativa del gobierno para obligar a las empresas de cualquier manera solo puede garantizar el fracaso del proyecto.
